# jafar-albadarneh/homebrew-tap

Homebrew tap for [Claude Switch](https://github.com/jafar-albadarneh/claude-switch).

## Install

```sh
brew tap jafar-albadarneh/tap
brew install --cask claude-switch
```

## Upgrade

```sh
brew upgrade --cask claude-switch
```

## What's in here

- `Casks/claude-switch.rb` — Cask formula for Claude Switch.

Claude Switch is an independent open-source project. It is not affiliated with, endorsed by, or sponsored by Anthropic.
